Interactive plots

This website contains unofficial data and figures that are under development. These data are unconfirmed an must not be used by any third parties. For official statistics and numbers please visit our website https://www.swisstransplant.org/.

The aim of this page is to visualize some of the organ donation and transplantation activity as interactive plots. You can click on the legend items to enable or disable categories, for example the organs. A double-click selects a single category.
